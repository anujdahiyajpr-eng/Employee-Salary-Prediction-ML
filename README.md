# Employee Salary Prediction using Machine Learning

## Overview
This project predicts an employee's salary based on Age, Experience, and Education Level using a Machine Learning model.

## Features
- Predict salary using Linear Regression
- Interactive Streamlit web application
- User-friendly interface
- Dataset stored in Amazon S3
- Source code hosted on GitHub

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- AWS S3
- Git & GitHub

## Project Structure
```
Employee_Salary_Prediction/
│── app.py
│── train_model.py
│── model.pkl
│── requirements.txt
│── README.md
└── data/
    └── salary_data.csv
```

## Machine Learning Algorithm
- Linear Regression
